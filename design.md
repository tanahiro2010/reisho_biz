# 冷笑.biz — デザイン仕様書

> Agentへの指示書。このドキュメントに従い静的HTMLサイトを実装すること。
> 解釈の余地があるときは「より冷笑に近い方」を選ぶこと。

---

## コンセプト

**「真顔で笑う」**

本物のコンサルティングファームのサイトと見紛うほどの真剣な作りでありながら、
読めば読むほどおかしい。そのギャップが冷笑感の正体。

- デザインは至って真面目
- コピーだけが静かにおかしい
- 「笑っている」とはどこにも書いていない

派手な演出は一切しない。飾りを足すのではなく、品のある佇まいの中にボケを忍ばせる。

## タイポグラフィ

### フォントの役割

| 役割 | フォント | 用途 |
|------|---------|------|
| 見出し (Display) | Noto Serif JP | H1、H2、屋号、セクションタイトル |
| 本文 (Body) | Noto Sans JP | 説明文、ナビ、フッター |
| ラベル (Utility) | `font-family: monospace` (system) | サービス番号、バッジ、注釈 |

### タイプスケール

```
Display  : 40px / weight 400 / line-height 1.3 / letter-spacing 0.04em
H1       : 28px / weight 400 / line-height 1.4
H2       : 20px / weight 400 / line-height 1.5
Body     : 15px / weight 400 / line-height 1.9
Small    : 12px / weight 400 / line-height 1.7
Label    : 11px / weight 400 / letter-spacing 0.12em / uppercase
```

**ポイント:** Noto Serif JPのweight 400は細くて品がある。700は使わない。

## コンポーネント詳細

### NAV

- 高さ: `64px`
- 左: 屋号「冷笑」をNoto Serif JP / 20px
- 右: テキストリンク 4項目（サービス / 所長について / お客様の声 / お問い合わせ）
- ボーダーボトム: `1px solid #D8D5CC`
- 背景: `#F7F5F0`
- スクロールで固定（`position: sticky; top: 0`）

### HERO

- 背景: `#F7F5F0`
- キャッチコピー: Display / Noto Serif JP
- レイアウト: テキスト左寄せ、右側は空白（詰め込まない）
- CTAは1つだけ（「まず相談してみる」など）

**コピー案（変更可）:**
```
あなたの悩み、
まず笑います。

貴方の状況を精査し、適切に冷笑した上で、
解決策を提示いたします。
```

### SERVICES

セクションラベル: `SERVICES` をLabelスタイルで左上に配置

3カラムグリッド、各カードの構造:

```
[番号 — monospace 11px]
[サービス名 — H2 / Serif]
[説明文 — Body / 2〜3行]
─────────────────────
[注釈 — Small / Mist色]
```

カード間はボーダー線（`1px solid #D8D5CC`）で区切る。カード背景は白。

**サービス内容:**

| # | 名前 | 説明 | 注釈 |
|---|------|------|------|
| 01 | 冷笑診断 | 現状をヒアリングし、まず笑います。その後、改善策を提示します。 | 所要時間：笑い終わるまで |
| 02 | 冷笑監査 | 計画・戦略を精査し、笑います。その後、レポートを納品します。 | 納品物：A4 2〜3枚程度 |
| 03 | 冷笑顧問 | 月次でお話を伺い、毎回笑います。継続的な冷笑をご提供します。 | 料金：お問い合わせください |

### VOICE（お客様の声）

引用符（`""`）を大きく表示するデザイン。Noto Serif JP。

```
"  [引用文]  "

      — [肩書き・匿名でよい]
```

**コピー案:**
- 「悩みが解決したかどうかはわかりませんが、笑われました。満足しています。」— 都内・30代
- 「こんなに堂々と笑われたのは初めてでしたが、気づいたら前に進んでいました。」— 関西・40代・個人事業主
- 「料金を聞いたら笑われました。それはちょっと違う気がします。」— 匿名

### PROFILE

2カラム（左1 : 右2）

**左カラム:**
- 円形アイコン（initials or プレースホルダー）
- 肩書き: `所長` — Label
- 名前: — Body / weight 500

**右カラム:**
- 挨拶文（Body / 4〜5行）
- 資格バッジ（横並び、monospaceフォント、枠線つき小タグ）

**バッジ例:**
- `冷笑力検定 一級`
- `優良冷笑事業所 認定`
- `笑顔で解決 3年連続受賞`

### CONTACT

```
まず、笑われてみませんか。

[お問い合わせはこちら →]
```

CTAボタン: アウトラインスタイル（背景なし、枠線のみ、hover時に背景Slate・文字白）

### FOOTER

```
冷笑                  © 冷笑 All rights reserved.
```

## インタラクション

- ホバー: テキストリンクはunderline、ボタンは背景反転
- スクロールアニメーション: **禁止**（静かに佇むサイトが正解）
- ローディング演出: **禁止**
- 動くものは一切置かない

## 実装仕様

- 静的HTML 1ファイル完結（`index.html`）
- CSSはHTMLファイル内の`<style>`に記述
- JavaScriptは不要（ナビのモバイルメニューのみ許可）
- Google Fonts CDN経由でNoto Serif JP / Noto Sans JPを読み込む
- 外部依存はフォントのみ
- レスポンシブ対応（ブレークポイント: `768px`）
  - モバイルではサービスカードを1カラムに変更
  - NAVメニューはハンバーガー化（シンプルで可）

---

## コピーライティング原則

1. **真顔で書く** — 「笑い」「ユーモア」「ジョーク」という単語は一切使わない
2. **受け流す** — 過剰な説明をしない。読者が気づくのを待つ
3. **丁寧語で通す** — フランクな表現はしない。品のある文体
4. **具体的に変** — 「笑い終わるまで」「A4 2〜3枚」など妙に具体的な数字を入れる
